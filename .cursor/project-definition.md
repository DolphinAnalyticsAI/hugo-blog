# Hugo Site File Structure Instructions

## Base File Structure
```plaintext

(root)/
├── src/                # Source files for Hugo
│   ├── archetypes/     # Templates for new content
│   │   └── default.md  # Default template for new content
│   ├── content/        # Site content (Markdown files)
│   │   ├── blog/       # Blog-specific content
│   │   └── pages/      # Additional sections (About, Contact, etc.)
│   ├── layouts/        # Templates for rendering content
│   │   ├── _default/   # Base templates (baseof.html, single.html, list.html)
│   │   ├── partials/   # Reusable components (header.html, footer.html)
│   │   └── blog/       # Custom blog templates
│   ├── static/         # Static assets (copied as-is to /public)
│   │   ├── css/        # Stylesheets
│   │   ├── js/         # JavaScript files
│   │   ├── images/     # Images/media
│   │   └── fonts/      # Fonts
│   └── config.toml     # Hugo configuration file
├── public/             # Output directory (auto-generated by Hugo)
└── resources/          # Auto-generated by Hugo for caching
```

### Create the directory structure
```bash
md src\archetypes & md src\content\blog & md src\content\pages & md src\layouts\_default & md src\layouts\partials & md src\layouts\blog & md src\static\css & md src\static\js & md src\static\images & md src\static\fonts
```

### Single line command to create the directory structure
